Верно ли, что для любых не диаметрально противоположных точек $A$ и $B$,
 а также им диаметрально противоположных $A'$ и $B'$:
\begin{enumerate}
    \item $AA'=BB'$;
    \item $AB=A'B'$;
    \item $AB'=A'B$;
    \item эти 4 точки лежат на одной прямой;
    \item $AB+BA'=\pi$;
    \item поляра точки $A$ совпадает с полярой $A'$.
\end{enumerate}
